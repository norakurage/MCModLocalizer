# MC Localizer

Minecraft Mod 向けの `en_us.json` を OpenAI API で日本語化 (`ja_jp.json`) する Flet 製 GUI アプリです。翻訳時に色コードやプレースホルダーを保護し、既存訳を壊さず差分のみを補完します。

## 主な機能
- Flet ベースのクロスプラットフォーム GUI（Windows/macOS/Linux）
- Mod JAR から `ja_jp.json` とリソースパックをワンクリック生成
- OpenAI Responses API を用いたバッチ翻訳（モデルは `gpt-4o-mini` などを指定可能）
- `%s`、`§a`、`{name}` のようなトークンを自動保護し、復元時に壊さない
- 既訳済みキーをスキップし、未翻訳分のみを処理
- CSV 用語集を読み込み、インストラクションに反映
- API キーを keyring（利用可能な場合）またはローカルストレージに安全に保存
- 実行ログ・進捗バー・停止ボタンで処理状況を可視化

## 必要条件
- Python 3.10 以上（Flet 0.28.3+ 推奨）
- OpenAI アカウントと API キー
- インターネット接続（OpenAI API 呼び出しに使用）

## セットアップ
1. 仮想環境の作成（任意）
   ```powershell
   python -m venv .venv
   .venv\Scripts\Activate.ps1
   ```
2. 依存パッケージのインストール
   ```powershell
   pip install flet openai keyring
   ```
   `keyring` は任意ですが、API キーを OS の資格情報ストアに保存する場合に必要です。

3. OpenAI API キーの設定
   - アプリの「設定」タブから入力し、保存先（keyring / local）を選択できます。
   - もしくは環境変数で指定します。
     ```powershell
     setx OPENAI_API_KEY "sk-..."
     setx OPENAI_MODEL "gpt-4o-mini"
     ```

## 実行方法
```powershell
python main.py
```
Flet がブラウザまたはネイティブウィンドウで UI を起動します。

## 使い方
1. **Mod JAR**：抽出したい Mod の JAR を指定します。
2. **出力フォルダ**：生成したリソースパックを置きたいフォルダを指定します（毎回自動的にサブフォルダを作成します）。
3. **用語集（任意）**：`用語,訳語` 形式の CSV を指定すると、AI への指示に組み込まれます。例：
   ```text
   Command Block,コマンドブロック
   Redstone,レッドストーン
   ```
4. **モデル選択**（設定タブ）：`gpt-4o-mini` / `gpt-4o` / `o4-mini` / `o4` から選択できます。環境変数 `OPENAI_MODEL` があれば既定で読み込みます。
5. **抽出 / ja_jp 生成** ボタンを押すと、
   - JAR から `assets/<modid>/lang/en_us.json` を一時フォルダに展開
   - OpenAI API で日本語化 (`ja_jp.json` を生成)
   - テンプレート（`a` フォルダ）を参考にリソースパックを組み立て、出力フォルダへ配置
   の一連処理を自動で実行します。進捗バーとログで状況を確認でき、途中で止めたい場合は **停止** を押します。

## トークン保護について
- `%1$s`、`§a`、`{player}`、`\n` など、ゲーム内で意味を持つ記法は翻訳前に `‹T0›` のような一時トークンに置き換えて保護します。
- 翻訳後に自動復元されるため、翻訳モデルがプレースホルダーを壊すリスクを抑えられます。
- Glossary を併用することで、用語統一も叩き込みやすくなります。

## ログと出力
- ログ欄には開始/停止、作業ディレクトリ、各バッチの進捗、警告やエラーが逐次表示されます。
- 処理が完了すると、一時フォルダは自動削除され、出力フォルダ直下に `<Mod名>_ja_resourcepack` が作成されます。
- `pack.mcmeta` は Java Edition 全バージョンをカバーする `supported_formats` 付きで出力されます。`pack.png` は自動生成しないため、必要に応じて手動で配置してください（既存の `pack.png` がある場合は自動で引き継ぎます）。
- 停止ボタンで中断した場合も、それまでに翻訳済みの部分は保存されます。

## トラブルシューティング
- **API キー未設定**：設定タブで保存し直すか、環境変数を確認してください。
- **用語集の読み込み失敗**：CSV にコメント行（`#`）を入れる場合は先頭に `#` を付け、カンマ区切りで 2 列にしてください。
- **Rate limit / ネットワークエラー**：OpenAI 側の制限で失敗する場合があります。一定時間後に再試行してください。
- **keyring が利用できない**：`keyring` のインストールや OS ごとのバックエンド設定が必要な場合があります。問題が続く場合は保存先を `local` に切り替えてください。

## 開発メモ
- Flet 0.28.3 以降の PascalCase API（`Icons` / `Colors` / `Alignment`）に合わせています。
- メイン処理はバックグラウンドスレッドで実行し、UI スレッドはログ更新のみ行います。
- OpenAI Responses API（`client.responses.create`）を利用しているため、`python-openai` v1 系が必要です。
